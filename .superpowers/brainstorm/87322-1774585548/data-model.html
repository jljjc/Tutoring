Data Model

Core tables in Supabase Postgres

users
id uuid PK
email text
role enum (student | parent)
full_name text
created_at timestamp
student_profiles
id uuid PK → users
parent_id uuid → users
year_level int (6)
school text
target_schools text[]
question_bank
id uuid PK
test_type enum (GATE | scholarship)
section text (verbal/maths/abstract)
topic text
difficulty int (1–5)
question_text text
options jsonb (A/B/C/D)
correct_answer text
explanation text
generated_at timestamp
test_sessions
id uuid PK
student_id uuid → users
test_type enum (GATE | scholarship)
started_at timestamp
completed_at timestamp
total_score int
projected_score float
test_answers
id uuid PK
session_id uuid → test_sessions
question_id uuid → question_bank
selected_answer text
is_correct bool
time_taken_secs int
tutoring_sessions
id uuid PK
student_id uuid → users
question_id uuid → question_bank
wrong_answer text
ai_explanation text
followup_question jsonb
mastered bool
attempts int

Key relationships: One parent → many students · One student → many test_sessions · One session → many test_answers · Wrong answers trigger tutoring_sessions · Question bank is shared + cached (generated by Claude, reused across students)
